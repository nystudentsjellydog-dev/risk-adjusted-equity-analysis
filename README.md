# Portfolio Management Analysis

## Overview
This project examines whether a stock portfolio built using financial indicators performs better than one built using intuition alone. The analysis was completed in R and focuses on portfolio construction, risk-adjusted return evaluation, and basic portfolio optimization.

## Project Objective
The project compares two stock portfolios:
- an intuition-based portfolio
- a data-driven portfolio built using financial indicators

The goal is to evaluate whether financial indicators can improve portfolio decisions and investment outcomes.

## Methods
The analysis includes:
- stock return correlation analysis for diversification
- Sharpe ratio calculation
- Jensen’s Alpha estimation using CAPM regression
- portfolio weight optimization by maximizing the Sharpe ratio
